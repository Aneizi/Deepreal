# Deepreal

This is a React/Vite app containing:

- Tailwind and Shadcn UI for styling
- [Gill](https://gill.site/) Solana SDK
- Shadcn [Wallet UI](https://registry.wallet-ui.dev) components

## Website
https://usedeepreal.com


## Run it yourself

### Installation
Download this repo

#### Install Dependencies

```shell
npm install
```

### Start the app

```shell
npm run dev
```
